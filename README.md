# Wuthering Wave Character Tracker
## Project Summary
This project is use to display the stats of a wuthering wave character instead of asking the other person about their characters stats. 
## Tech Stack
-React
-Typewscript
-Tailwind CSS
-vite
## How to use
There are three .ts files within the data folder. Please Edit them to match your account stat as I have yet to implement an automated system to do so. 
## Further Note
So far, this is only usable for my own purpose.
